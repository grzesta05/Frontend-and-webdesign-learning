[Spis zawartości - Rodzaje matryc w TV]

  • Matryca LED
  • Matryca QLED
  • Matryca plazmowa

Matryca LED [SPIS]

Matryca LED (Light Emitting Diode), jak sama nazwa wskazuje, jest zbudowana z diod elektroluminescencyjnych – w celu uzyskania szerokiej palety kolorów wykorzystywane są trzy kolory: czerwony, zielony i niebieski, które tworzą piksel. W zależności od stopnia podświetlenia pozwalają na stworzenie efektu milionów różnych odcieni danego koloru

Matryca QLED [SPIS]

Telewizory QLED stanowią dobrą alternatywę dla modeli wyposażonych w matrycę QLED. W panelu LCD stosowane są kropki kwantowe, które pochłaniają, a następnie emitują światło w innym kolorze. Producent za przykład podaje kropki o wielkości 2 nm, które emitują światło niebieskie. Kropki 6-7 nm są w stanie wyświetlać światło czerwone.

Matryca plazmowa [SPIS]

Ksenon i neon, gaz stosowany w wyświetlaczach plazmowych, zawarty jest w setkach tysiącach drobnych komórek umieszczonych pomiędzy dwiema szklanymi płytami, między którymi umieszczono także elektrody adresujące, rozciągające się na cały ekran. Jeden zestaw elektrod umieszczono w liniach poziomych, a drugi w liniach pionowych tworząc swego rodzaju siatkę.

= User Action Pseudo-Class Selectors = | :hover | :focus | :active | :focus-within |

Pseudoklasa: :hover{ ... }, :focus{ ... }, :active{ ... }

Kliknij przycisk potem nacisnij tab by zobaczyć stan:focus. Następnie kliknij i przytrzymaj PPM by zobaczyć stan :active.

Pseudoklasa :focus-within { ... }

:focus-within aktywuje się, kiedy element, na którym działa jest w stanie :focus lub jakikolwiek jego potomek jest w stanie :focus.

Twój unikatowy numer:

Wartość parametru n może być:

- wyrażeniem (odd lub even) - parzyste lub nieparzyste;

- numerem (1, 2, ...);

- prostą formułą (2n, 2n+1, -n).

:nth-last-child(n)

Pseudoklasa działa podobnie jak :nth-child(n)

z tym że elementy liczone są od ostatniego wystąpienia.

tr:nth-child( wartość ) zaznaczy ? rząd w tabeli ?

tr:nth-child(0n+1) n startuje od 0 z przyrostem o 1.

jeżeli n=0, to 0 * 0 + 1 = 1. pierwszy rząd;

jeżeli n=1, to 0 * 1 + 1 = 1. pierwszy rząd;

jeżeli n=2, to 0 * 2 + 1 = 1. pierwszy rząd;

tr:nth-child(2n+1)

jeżeli n=0, to 2 * 0 + 1 = 1. pierwszy rząd;

jeżeli n=1, to 2 * 1 + 1 = 3. trzeci rząd;

jeżeli n=2, to 2 * 2 + 1 = 5. piąty rząd;

tr:nth-child(2n)

jeżeli n=0, to 2 * 0 = 0. brak rzędu;

jeżeli n=1, to 2 * 1 = 2. drugi rząd;

jeżeli n=2, to 2 * 2 = 4. czwarty rząd;

tr:nth-child(-n+3) 3 pierwsze rzędy

jeżeli n=0, to 0 + 3 = 3. trzeci rząd;

jeżeli n=1, to 0 + 2 = 2. drugi rząd;

jeżeli n=2, to 0 + 1 = 1. pierwszy rząd;

= Tree-Structural - -Child-Indexed- - Pseudo-Class Selectors =

:first-child | :last-child | :only-child | :nth-child(n) | :nth-last-child(n)

Pseudoklasa: :first-child { ... }

Pseudoklasa :first-child sprawia iż własności stosowane są do elementu tylko wtedy gdy jest on pierwszym węzłem potomnym swojego rodzica. Kolejne dwie pseudoklasy działają analogicznie.

Pseudoklasa: :nth-child(n) { ... }

Dopasowuje elementy na podstawie ich pozycji w grupie rodzeństwa.
